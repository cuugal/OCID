Search Chemicals
To Search for a Chemical:
1. Enter the Chemical Name,
2. select a Location,
3. then click 'Search'.

The chemical's CAS # number can be used to search instead of the Chemical Name.

The wildcard character * may be used in the search text to match inexact chemical names.
    -- An asterisk(*) matches zero or more characters ( eg ben* matches benzene and benzoic acid).
    -- Use * to search ALL CHEMICALS at a particular location.
    -- Using *chemical name* is a good way to search a chemical name, as it accounts for variations in naming.

Checking the "Sort by name" box orders the inventory by name rather than by location then name. The default, sorting by location, is useful when performing stocktakes.
