OCID Instructions
For Administrator

LOGIN

Different login ID's have different access privileges to the system. Your login will allow you access to the whole system. In the Admin section you can create locations and update information about a location, such as changing manager names and passwords.

 

Once logged in, the black bar at the top of the screen shows four options

  • Search chemicals
  • Update & Add
  • Risk Assessment
  • Admin

SEARCH CHEMICALS

Not applicable.

UPDATE AND ADD

If you need to delete all the chemicals within a location without deleting the location itself, you can manually delete all the chemicals at a location in the 'update chemicals at a location' section. Alternatively the location can be deleted in the Admin section and then recreated.

RISK ASSESSMENTS

Not applicable.

ADMIN

In 'Update the Location', all details about a particular location can be edited.

In 'Add New Location', a new location can be created for an existing manager or a new manager and location can be added.
